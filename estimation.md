# Time Estimation for the h2go project

## Summary 
it'll take about 14-16 days to be completed

## Estimation Break Down

### 1. Env Setup / intial Design (1-2 days)
- DB setup
- setup the dev environment
- Think through flows and user stories 
- initial class diagram

### 2. Auth / User Management (2-3 days)
- Authentication 
- Authorization
- email verification
- User management (profile/crud)

### Provider Management (1 day)
- Provider Registeration
- Admin approval
- provider Management(profile/crud)

### Product Management (2 days)
- Provider adds products
- update price and stock
- user views providers/products

### Subscription Management (1 day)
- user subscribes
- provider approves/reject
- cancel Subscription (user/provider)

### Order Management (3 days)
- create order (one time/ recurring)
- manage order status
- order-product relation
- provider approval

### Admin (1 day)
- view users
- Remove provider
- approve provider (done in provider management)

### Testing (2 days)
- unit tests
- Manual tests

### documentation (1 days)
- README
- update docs (if needed)
